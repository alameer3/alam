# موقع أكوام - المكتبة الترفيهية العربية

## نظرة عامة على المشروع

موقع أكوام هو منصة ترفيهية عربية متكاملة لتصفح ومشاهدة الأفلام والمسلسلات والبرامج التلفزيونية العربية والأجنبية. تم تطوير هذا المشروع باستخدام تقنيات حديثة لتوفير تجربة مستخدم متميزة.

## تفاصيل المشروع

### الهدف الرئيسي
إنشاء موقع ترفيهي عربي يحاكي تصميم ووظائف الموقع الأصلي ak.sv مع:
- واجهة مستخدم عربية حديثة ومتجاوبة
- نظام تصفح سهل ومنظم للمحتوى
- دعم البحث المتقدم
- تصميم يركز على تجربة المستخدم العربي

### المميزات الأساسية
- **الصفحة الرئيسية**: عرض المحتوى المميز والأحدث
- **تصفح الأفلام**: قسم مخصص لعرض الأفلام بتصنيفات مختلفة
- **المسلسلات**: قسم للمسلسلات العربية والأجنبية
- **البرامج التلفزيونية**: برامج متنوعة ومنوعات
- **نظام البحث**: بحث متقدم عن المحتوى
- **واجهة متجاوبة**: تدعم جميع الأجهزة

## البنية التقنية

### التقنيات المستخدمة
- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **UI Components**: shadcn/ui
- **State Management**: Zustand
- **Data Fetching**: TanStack Query
- **Routing**: Wouter
- **Styling**: Tailwind CSS مع دعم RTL

### هيكل المشروع
```
project/
├── client/          # تطبيق React الأمامي
│   ├── src/
│   │   ├── components/  # المكونات المعاد استخدامها
│   │   ├── pages/       # صفحات التطبيق
│   │   ├── lib/         # مكتبات مساعدة
│   │   └── hooks/       # React Hooks مخصصة
├── server/          # خادم Express
│   ├── routes/      # مسارات API
│   ├── middleware/  # وسطاء الأمان والتحقق
│   ├── services/    # خدمات التطبيق
│   └── database/    # إدارة قاعدة البيانات
├── shared/          # أنواع البيانات المشتركة
└── serverdata/      # بيانات وصور الخادم
```

## الصفحات المتاحة

### الصفحة الرئيسية (/)
- عرض الشعار والبحث الرئيسي
- أقسام التصفح (أفلام، مسلسلات، برامج، منوعات)
- المحتوى المميز والحديث
- إحصائيات الموقع
- التذييل بالروابط المفيدة

### صفحة الأفلام (/movies)
- عرض قائمة الأفلام المتاحة
- تصفية حسب النوع والجودة
- ترتيب حسب التقييم والتاريخ

### صفحة المسلسلات (/series)
- عرض المسلسلات المتاحة
- تصنيف حسب النوع (عربي، تركي، أجنبي)
- معلومات الحلقات والمواسم

### صفحة البرامج (/shows)
- البرامج التلفزيونية والرياضية
- تصنيف حسب النوع والمحتوى

### صفحة المنوعات (/mix)
- محتوى متنوع (ألعاب، تطبيقات، مسرحيات)
- تصنيف خاص بالمحتوى الترفيهي

## نظام البيانات

### نموذج البيانات الأساسي
```typescript
interface ContentItem {
  id: number;
  title: string;
  titleAr: string;
  type: 'movie' | 'series' | 'show' | 'mix';
  poster: string;
  rating: number;
  releaseDate: string;
  quality: string;
  country: string;
  genres: string[];
  description: string;
}
```

### نظام التخزين
- **MemStorage**: تخزين مؤقت في الذاكرة للتطوير
- **Database**: قاعدة بيانات JSON للإنتاج
- **Images**: ملفات الصور في مجلد serverdata/images
